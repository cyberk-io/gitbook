## I. Landing Page Design

- [ ] **Desktop Version (1440px)**
  - [ ] Header (Logo, Menu, CTA)
  - [ ] Hero Section (Headline, Sub-headline, Hero Image/Illustration, CTA)
  - [ ] Features/Benefits Section
  - [ ] About/Story Section
  - [ ] Footer (Links, Socials, Copyright)
  - [ ] *Các section khác theo yêu cầu:* ...................................................
- [ ] **Mobile Responsive (375px/390px)**
  - [ ] Menu chuyển sang dạng Hamburger/Drawer
  - [ ] Font size và Spacing được tối ưu cho cảm ứng
  - [ ] Layout xếp chồng (Stacking) hợp lý
- [ ] **SEO & Social Standards**
  - [ ] **Open Graph Image**: Ảnh đại diện chuẩn kích thước (1200x630px) hiển thị khi chia sẻ link website lên Facebook, Zalo, LinkedIn.
  - [ ] **Favicon**: Icon nhỏ trên tab trình duyệt (các size: 16x16, 32x32, 192x192).
  - [ ] **App Icon (nếu là App)**: Icon ứng dụng chuẩn kích thước cho iOS/Android.
