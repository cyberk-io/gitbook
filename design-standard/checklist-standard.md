# CHECKLIST DỰ ÁN THIẾT KẾ - STANDARD PACKAGE
*Dành cho dự án vận hành hoàn chỉnh, bao gồm marketing cơ bản.*

> **Quy định**:
> - Team dự án copy file này vào thư mục dự án hoặc tạo Checklist tương ứng trên Figma (Trang Cover).
> - Đánh dấu [x] khi hoàn thành.
> - CEO/PM kiểm tra dựa trên checklist này để nghiệm thu.

## I. Design System (UI Library) - **QUAN TRỌNG**
- [ ] **Typography System**: Khai báo đầy đủ H1-H6, Body, Caption, Button Text.
- [ ] **Color Palette**: Primary, Secondary, Neutral, Semantic (Success, Warning, Error, Info).
- [ ] **Grid & Spacing System**: Quy định lưới và khoảng cách (4px/8px grid).
- [ ] **Component Library (Variants & Auto Layout)**
  - [ ] Buttons (Primary, Secondary, Ghost, Disabled...)
  - [ ] Inputs (Default, Focus, Error, Disabled)
  - [ ] Dropdowns / Selects
  - [ ] Cards / Containers
  - [ ] Modals / Popups
  - [ ] Navigation / Sidebar
- [ ] **Icon Set**: Bộ icon đồng bộ (Outline/Solid).

## II. Landing Page Design (Đồng bộ Design System)
- [ ] **Desktop Version** (Full sections)
- [ ] **Mobile Responsive** (Optimized UX)
- [ ] **Tablet Responsive** (Optional - nếu yêu cầu)
- [ ] **Interaction**: Có note về hiệu ứng Hover, Click, Transition cho Dev.

## III. Application Design (Đồng bộ Design System)
- [ ] **Full Flows**: Thiết kế đầy đủ các luồng nghiệp vụ (Happy case & Edge cases).
- [ ] **Interactive Prototype (Cơ bản)**: Link các màn hình chính để demo luồng.
- [ ] **States**: Hover, Active, Focus, Disabled, Loading, Error, Empty, Success.

## IV. Social Media Kit (Marketing Assets)
- [ ] **Avatar**: File xuất cho Facebook, LinkedIn, Twitter (đúng kích thước).
- [ ] **Cover Image**: Ảnh bìa đồng bộ thương hiệu cho các kênh social.
- [ ] **Open Graph Image**: Ảnh hiển thị khi share link website.
- [ ] **Post Templates (03 mẫu)**
  - [ ] Template 1: Công bố ra mắt (Launching)
  - [ ] Template 2: Giới thiệu tính năng (Feature Highlight)
  - [ ] Template 3: Trích dẫn/Feedback (Quote)

## V. Video Teaser
- [ ] **Video Intro (15-30s)**
  - [ ] Kịch bản/Storyboard được duyệt.
  - [ ] Motion Graphic mượt mà.
  - [ ] Nhạc nền phù hợp (bản quyền hoặc miễn phí).
  - [ ] Xuất file MP4 Full HD.

## VI. General
- [ ] **Clean Handoff**: File Figma sạch sẽ, có trang "Ready for Dev".
